About HACOM Project

HACOM is a graduation project that focuses on real-time data processing and analysis for e-commerce recommendations. The project combines modern technologies including:

  • Apache Spark for data processing
  • Kafka for real-time data streaming
  • Redis for caching and real-time data storage
  • Flask for backend services
  • Modern web technologies for frontend
  • Gemini for enhanced search capabilities

Key Features

  • Smart Product Recommendations: Get personalized product suggestions based on your shopping behavior and preferences
  • Real-time Updates: Recommendations update instantly as you browse and shop
  • Intelligent Search: Find products easily with our advanced search system that understands what you're looking for
  • Multiple Recommendation Types: Discover products through various methods like similar items, popular choices, and personalized picks
  • Fast and Reliable: Enjoy quick response times and smooth shopping experience
